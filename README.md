## Performance Profiles

Choose a profile in shader settings that matches your hardware:

| Profile | FPS (iGPU) | FPS (dGPU) | Description |
|---------|------------|------------|-------------|
| **ULTRA_PERFORMANCE** | 150+ | 300+ | Maximum FPS, minimal effects |
| **PERFORMANCE** | 100-120 | 200+ | Good balance for iGPU |
| **BALANCED** | 60-80 | 120-180 | Recommended for most users |
| **QUALITY** | 40-60 | 90-120 | More effects, needs dGPU |
| **ULTRA_QUALITY** | 30-45 | 60-90 | All features, best visuals |

## Installation

1. Install [Iris Shaders](https://irisshaders.dev/) or OptiFine
2. Download Wispy Shaders
3. Place the ZIP file in `.minecraft/shaderpacks/`
4. Select Wispy in Video Settings â†’ Shaders
5. Choose a performance profile

## Optimizations

- Disabled expensive composite passes by default
- Fast gamma correction
- Simplified post-processing
- Optimized fog and sky rendering
- Reduced shader complexity
